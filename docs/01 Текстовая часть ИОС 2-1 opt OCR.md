3) Описание системы автоматизации водоснабжения.

Для жилого дома проектом предусмотрено автоматическое, дистанционное и ручное
управление водоснабжением при пожаре. На прибор «Сигнал-20» выведены

обратные сигналы о срабатывании водоснабжения при пожаре.

На блок индикации С2000-БИ по линии интерфейса собирается вся информация о
работе водоснабжения при пожаре. Режим местного управления предназначен только
для проведения ремонтных и пусконаладочных работ.
Для управления пожарными насосами в проекте использован шкаф управления
«Грантор» типа АЭП, обеспечивающий:
- ручной (для опробования) режим работы;
- автоматическое включение резервного насоса при аварии рабочего;

- дистанционное включение от кнопок, установленных в шкафах пожарных кранов на

каждом жилом этаже дома,

- защиту насосов от «сухого хода» (при отсутствии воды на входе).
